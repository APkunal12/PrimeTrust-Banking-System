# 🏦 PrimeTrust — Secure Banking System

PrimeTrust is a full-stack banking web application that simulates real-world banking operations in a secure and user-friendly environment. It allows users to manage accounts, perform transactions, and monitor financial activities efficiently.

---

## 🚀 Features

### 👤 User Features
- Secure user registration and login
- Account dashboard with balance overview
- Deposit and withdrawal functionality
- Fund transfer between accounts
- Transaction history tracking
- Profile management
- Password change option

### 🏦 Banking Operations
- Account creation
- Balance management
- Transaction recording
- Real-time balance updates

### 🛡️ Security Features
- Authentication and session management
- Password hashing
- Input validation
- Protection against unauthorized access

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/PrimeTrust-Banking-System.git
cd PrimeTrust-Banking-System
```

### 2. Create virtual environment

```bash
python -m venv venv
source venv/bin/activate     # Mac/Linux
venv\Scripts\activate        # Windows
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Apply migrations

```bash
python manage.py migrate
```

### 5. Run the server

```bash
python manage.py runserver
```

### 6. Open in browser

```
http://127.0.0.1:8000/
```
